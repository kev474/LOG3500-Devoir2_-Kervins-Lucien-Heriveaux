# Description

Ce projet a été réalisé dans le cadre du cours LOG3500 - Conception et programmation de sites Web I.

Il s'agit d'une application web météo permettant de rechercher une ville et d'afficher des informations météorologiques à l'aide d'une API.

# Fonctionnalités

- Interface adaptée aux appareils mobiles et ordinateurs
- Recherche d'une ville
- Affichage de la température
- Affichage de la localisation
- Affichage des conditions météorologiques
- Affichage de la vitesse du vent

# Technologies utilisées

- HTML5
- CSS3
- JavaScript
- API météo Open-Meteo
